## Using PRDs in Weekly Meetings

The weekly scrum meeting sheet includes a "PRD Progress" section for each app:
- **Where we are** -- current implementation state
- **Where we want to be** -- next milestone from the PRD
- **% complete** -- rough progress
- **What's blocking us** -- impediments

This ensures weekly discussions are anchored to the product vision, not just recent commits.
